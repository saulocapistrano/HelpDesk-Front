Olá! Seja bem vindo ao sistema Help Desk
Esse é um sistema para gerenciamento de chamados desenvolvido em Angular 12 e Spring Boot 2.
O sistema está rodando na nuvem sendo o front na Vercel e o back na Heroku.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT entre outas skills. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.

Fique a vontade para testar o sistema e deixar sua sugestão.
